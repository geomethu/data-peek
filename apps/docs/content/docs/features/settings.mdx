# Settings

Open Settings with `Cmd+,` (macOS) or `Ctrl+,` (Windows/Linux).

## Available Settings

### Editor

- **Hide query editor by default** - Table preview tabs start with the editor collapsed
- **Hide quick query panel** - Hides the mini SQL editor in the sidebar

### Results

- **Default page size** - Number of rows per page: 25, 50, 100, 250, or 500
- **Expand JSON by default** - Auto-expand JSON/JSONB cells in results
  - **JSON expand depth** - How many levels deep to expand (1-5)

### Query

- **Query timeout** - Maximum execution time in seconds (0 = no timeout)

### Fun

- **Pokemon Buddy** - Enable the Pokemon companion in the sidebar

### Reset

Click **Reset to Defaults** to restore all settings to their original values.

<Callout type="info">
  Settings are stored locally and persist across app restarts.
</Callout>
